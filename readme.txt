The PO_SMIP package contains the following files: 
-----------------------------------------------
Prediction and optimization of seismic metamaterial isolation performance for building structures contains the following six sections：
1. Hard scatterer IBEM procedure：Used to form multi body scattering effects, where the number and position of spheres need to be changed according to user needs.
2. FMM acceleration program：Origin code from https://doi.org/10.1016/j.enganabound.2005.11.006
3. Ground motion feature extraction：Extracting Ground Motion Features
4. Frequency domain conversion program：Frequency-time domain interpolation using discrete Fourier integration
5. SDOF generation program：Dynamic response calculation of a SDOF using Duhamel integral
6. PSO-ANN prediction program：Implementing reverse prediction function using PSO accelerated ANN
